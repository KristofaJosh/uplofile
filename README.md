# Uplofile (monorepo)

[![Beta](https://img.shields.io/badge/status-beta-yellow.svg)](https://uplofile.kristofajosh.dev)

Composable file upload components for React.

> Beta: The library is in active development. Things may change — your feedback is super helpful!

This repository contains:
- packages/uplofile — the Uplofile React library published to npm
- app/doc — the documentation/demo app
